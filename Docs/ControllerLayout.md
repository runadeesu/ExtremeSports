# Controller Layout

## Controller Required
Extreme Sports requires a compatible game controller. Keyboard and mouse are not supported for gameplay.

## Supported Controllers
- Xbox Controller
- PlayStation DualSense / DualShock 4
- Generic XInput Controllers

## Gameplay
- Left Stick: Body Weight / Movement
- Right Stick: Board / Scooter Deck / Bike Control
- LT / RT: Modifiers
- LB / RB: Trick Modifiers

## Stance
### Skateboard
- Regular
- Goofy

### Scooter
- Regular
- Goofy

### BMX
- Standard
